# Учебные проекты (Яндекс Практикум)

Часть работ, которые сделал во время учебы на Я.Практикуме.

## Проекты
- `async-yacut` — асинхронный сокращатель ссылок на Flask + SQLite, выдаёт короткие URL, редиректы и сохраняет/загружает файлы на Я.Диск.
- `foodgram` — сервис "cоцсеть" рецептов: публикации, избранное, покупки; стек Django + DRF + React, прод-сборка в Docker.
- `kittygram_final` — "соцсеть" для достижений котов: Django REST API, React-фронт, CI/CD через GitHub Actions и Docker Hub.
- `the_snake` — аркада "Змейка" на Pygame, запуск через `python the_snake.py`.

Подробнее — в README внутри каждого каталога.
