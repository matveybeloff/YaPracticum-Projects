# the_snake

Первый проект. Небольшая аркада «Змейка» на Pygame: собирайте яблоки, растите хвост, следите за счётом и рекордом.

## Как запустить
- Установите зависимости: `pip install -r requirements.txt`
- Запустите игру: `python the_snake.py`
- Управление стрелками; счёт и рекорд отображаются под полем.
